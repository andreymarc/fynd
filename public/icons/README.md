# App Icons

This directory should contain PWA icons in the following sizes:

- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels)
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels) - iOS
- icon-192x192.png (192x192 pixels) - Android
- icon-384x384.png (384x384 pixels)
- icon-512x512.png (512x512 pixels) - Splash screen

## Creating Icons

You can create these icons from your logo.jpg:

1. Use an online tool like https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator
2. Or use image editing software to resize logo.jpg to each size
3. Icons should be square PNG files with transparent or solid backgrounds
4. For best results, use a simple icon design that works at small sizes
