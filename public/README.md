# Public Assets

Place your static assets here, including:

- `logo.jpg` - Your Fynd logo (REQUIRED - place it here!)
- Other images, fonts, etc.

## Logo Setup

1. **Save your logo file** as `logo.jpg` in this directory (`/public/logo.jpg`)
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.svg`
3. Recommended size: at least 200px height for high-quality display
4. The logo will automatically appear in the navbar once placed here
